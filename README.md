### Getting Started

Clone:

```
git clone https://github.com/KleoHasani/StickyNote.git
```

Install dependencies:

```
npm install
```

Start:

```
npm start
```

### Required

Electron v9.1.2

Electron Builder v22.9.1

Node v12.19.0

NPM v6.14.7

### Build With

Electron v11.3.0

Electron Builder v26.6.3

Build:

```
npm run dist
```

Windows:

- dist/_installer_.exe

Debian:

- dist/_installer_.deb

- dist/_installer_.AppImage
